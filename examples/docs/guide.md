### 安装

推荐使用 `npm` 或 `yarn` 进行安装，它能更好地和 `webpack` 打包工具配合使用。而且可以更好的和 `es6` `typescript` 配合使用。并且支持按需引入

```shell
npm i yunxiu-next -S
# or
yarn add yunxiu-next
```

如果您了解 node.js、npm 安装，并希望配合 webpack 使用，请阅读下一节：[快速上手](/#/start)。
