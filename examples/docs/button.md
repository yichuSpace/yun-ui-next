## Button 按钮

常用的操作按钮。

### 基础用法

默认按钮，可设置`dashed`虚线和`background`拟物按钮

::: demo

```html
<template>
  <div class="demo-button" flex="cross:center">
    <yun-button>Default</yun-button>
    <yun-button dashed>Dashed</yun-button>
    <yun-button background>Background</yun-button>
  </div>
</template>
```

:::
